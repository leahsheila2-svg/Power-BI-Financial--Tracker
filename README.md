# Power-BI-Financial--Insight-Dashboard
This powerBI project analyzing uber spending across department that is HQ and Properties to increase cost visibility and accountability. 
# Project objectivity
The main aim of this dashboard is to visualize and compare departmental uber spending, identify top-spending department and highlight cost pattern between HQ and Property. 
# Key Feature
- Departmental-level uber usage and spending analysis
- HQ vs Property comparision
- Top department spender and trip frequency insight
- Interactive visuals for time, cost and depratment filter
  ## Tools Used
  Power Bi
  Power Query (Data cleaning)
  DAX Function
  Excel
# Insight
This dashboard helps the management to track transport cost, enforce travel policy, and identify opportunities for cost reduction
